# a11yhawk

Open-source, self-hostable web accessibility scan engine. Playwright page capture, Lighthouse accessibility audits, and bring-your-own-key LLM analysis produce structured WCAG reports (JSON, markdown, and a self-contained HTML page). Runnable as a Node library, an `npx` CLI, or a Docker microservice on your own infrastructure. No accounts, no telemetry, no external services beyond the LLM endpoint you configure.

## Status

**Under active development.** This version is a name-reserving stub and contains no functionality. Watch this repo for the first working release.

## Background

a11yhawk is the open-source engine behind [AccessHawk](https://accesshawk.ai), extracted and maintained by the same author so that teams can run the scan pipeline entirely on their own infrastructure.
